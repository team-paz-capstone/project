### Server

##### Connect to the server
Run this before launching the server.
```bash
heroku config -a pazcapstone
```

##### Logging into the server
If you need to log in to the server for
local queries:
```bash
heroku pg:psql -a pazcapstone
```

#### Run the server

```bash
mvn spring-boot:run
```
